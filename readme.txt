The easiest way to install Unique Chat on your WordPress site! 

== Description ==

This plugin makes it easy to install and use Unique Chat on your WordPress site. **Please note:** You will need to have an active Unique Chat account to make use of this plugin.

= Visit our site for more info: =

**[Unique Chat](https://uniquelibrary.com/patron-services/live-web-chat-software/)**

== Installation ==

= Automatic Installation: =

* You can install this plugin directly from your WordPress dashboard's 'Plugins' section.

= Manual Installation: =

* Unzip contents of this .zip file to your `/wp-content/plugins/` directory.

= After Installation: =

1. Activate the plugin through the 'Plugins' section in WordPress.
2. Go to `Settings >> Unique Chat SE`
3. Check enable. Type in your [Unique Chat Widget ID]. Click save.
4. Check with your Unique Chat service representative to see if the widget should be active.
5. Once active, visit your site to see a chat icon!
